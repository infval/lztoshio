# lztoshio
Compression tool for Toshio Toyota (Sega Mega Drive) games

**USAGE FOR DECOMPRESSION**: lztoshio.exe [Filename] d [HexOffset]

USAGE FOR COMPRESSION: LZToshio.exe [InFilename] [OutFilename] c

List of games:
 - Crusader of Centy (Soleil);
 - Earnest Evans;
 - Final Zone;
 - Granada;
 - El Viento;
 - Ranger-X;
 - Zan Yasha Enbuden.
